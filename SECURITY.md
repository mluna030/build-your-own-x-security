# Security Policy

This repository is a curated list of links. It ships no executable code to readers; the only automation is link checking and pull-request validation that runs in CI.

## Reporting a problem

Use one of these, depending on what you found:

- A malicious, harmful, or hijacked link in the list: open a private security advisory from the Security tab, or open a normal issue without sharing exploit detail, and it will be removed.
- A problem with this repository's GitHub Actions or automation: report it privately via a security advisory.

Please do not open public issues that include working exploit payloads.

## Scope and trust

The linked tutorials are third-party content hosted elsewhere. Links are checked for being live and on-topic, but the destination sites are not controlled or audited here, and their content can change at any time. Verify anything before running it.

All offensive-security material in this list is for authorized testing, CTFs, and defensive learning only. Run it solely against systems you own or are authorized to test.
