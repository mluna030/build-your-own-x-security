# Contributing

This list only accepts tutorials that teach building a thing from scratch, with real, complete code. Tool walkthroughs, thin listicles, and marketing pages are out of scope.

## Adding an entry

1. Open an issue or a pull request with the link.
2. An entry qualifies when it is live and freely accessible, genuinely teaches building the thing from scratch, and includes real, complete code.
3. Add it to the matching section in README.md, grouped by language, using the entry format below.

## Entry format

    * [**Language**: _Title_](https://example.com)

Bold the language, italicize the title, matching the build-your-own-x style. Group entries by language within a section, and use the author's exact title.

## Scope

Offensive projects must be framed for authorized testing, CTFs, and defensive learning, and should reference running against targets the reader controls. Anything aimed at unauthorized intrusion, mass targeting, or detection evasion for real attacks is rejected.
